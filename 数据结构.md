# (一)链表

## 链表的定义

物理顺序：即按从前到后的顺序依次排序，如排队的第一个人，第二，第三.........

逻辑顺序：即按照一定规律进行排序，如：赵，钱，孙，李，周，吴.........（按人数多少排序）

与顺序表存储数据的方式不同（顺序表是把逻辑位置上相邻的数据，存储在物理位置上相邻的位置），链表是物理位置上随机存储数据，不一定按逻辑顺序排序。

## 基本结构，及特点

每个节点由两部分组成：![1](E:\照片\1.png)   数据域存储该地址的数值，指针域存储下一个元素所在地址

 双链表（两个指针域![3](E:\照片\3.png))： 

循环链表：尾结点指向首结点 

空链表：前一位指针域为空

### 总结

链表：顺序读取，随机存储。即存数据时是随机的，而读取时只能一个一个找过去不能跳

顺序表：随机读取，顺序存储

# (二)栈,队列

## 1.1栈和队列特点和定义

他俩其实都是顺序表的子集，是顺序表的特殊形式

栈：先进后出，只允许在栈的最后(即栈顶)进行操作。

队列：先到先出，只允许在表头删除，在表尾插入。

## 1.2栈的初始化

## 1.2二叉树遍历

树下还是树

先序(根左右):先访问树的根，再先序遍历左子树，然后先序遍历右子树		

中序(左根右)：中序遍历左子树，访问树的根，中序遍历右子树

int bubbleSort1(int length,int *a) {
    int flag=1;
    while (flag == 1) {//这第一个循环是用来控制排序次数
        flag = 0;
        for (int i = 0; i < length - 1; i++) {//第二个循环是用来在数组里两两比较大小的比较
            if (a[i] < a[i + 1]) {
                int temp = a[i];
                a[i] = a[i + 1];
                a[i + 1] = temp;
                flag = 1;
            }
        }
    }
    return 0;
}

int bubbleSort(int length,int *a) {
    int flag = 1;
    while (flag == 1) {
        flag = 0;
        for (int i = 0; i<length-1; i++) {
            if (a[i] < a[i + 1]) {
                int temp = a[i+1];
                a[i + 1] = a[i];
                a[i] = temp;
                flag = 1;
            }
        }
    }

int searchNumber(int *a,int number,int length) {
    int first = 0;
    int last = length-1;//这边是数组越界问题
    int middle = (first + last) / 2;
    while (first<=last) {//是否需要等号的问题
        middle = (first + last) / 2;//数值溢出问题；优化：middle=first+(last-first)/2
        if (a[middle] == number) return middle;
        else if (a[middle] < number)first = middle + 1;//边界收敛问题：若此处只设first=middle 则会出现没有跳过已经确定不符合条件的middle
        else if (a[middle] > number)last = middle - 1;
