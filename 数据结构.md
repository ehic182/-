# (一)链表

## 链表的定义

物理顺序：即按从前到后的顺序依次排序，如排队的第一个人，第二，第三.........

逻辑顺序：即按照一定规律进行排序，如：赵，钱，孙，李，周，吴.........（按人数多少排序）

与顺序表存储数据的方式不同（顺序表是把逻辑位置上相邻的数据，存储在物理位置上相邻的位置），链表是物理位置上随机存储数据，不一定按逻辑顺序排序。

### 总结

链表：顺序读取，随机存储。即存数据时是随机的，而读取时只能一个一个找过去不能跳

顺序表：随机读取，顺序存储

# (二)栈,队列

## 1.1栈和队列特点和定义

他俩其实都是顺序表的子集，是顺序表的特殊形式

栈：先进后出，只允许在栈的最后(即栈顶)进行操作。

队列：先到先出，只允许在表头删除，在表尾插入。

int main() {//应用
	SElemType E;//定义插入元素
	linkstack L;//定义头指针，它指向头节点
	initallinkstack(&L);//为头指针指向的头结点初始化，注意：一定要先初始化，再读入值否则会出现野指针
	scanf_s("%c", &E);
	push(&L, E);//入栈
	printf("%c\n", L->next->data);
}
